El equipo desarrollador de aplicaciones para m\u00f3viles SEGA\u2019s Miku Flick publico en\u00a0Twitter el d\u00eda de hoy anunciando que el servicio dejara de funcionar para la aplicaci\u00f3n Miku Flick\/02, uno de sus juegos de ritmo para dispositivos iOS. El tweet indica que el servicio del juego terminara a finales de este mes el 19 de Julio. Las razones de la discontinuaci\u00f3n involucra incompatibilidades con los nuevos lanzamientos de dispositivos, cambios en las APIs de facturaci\u00f3n de Apple, y reorientaci\u00f3n de los esfuerzos del equipo de SEGA.<\/p>\n
