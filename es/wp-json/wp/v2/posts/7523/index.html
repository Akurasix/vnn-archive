De acuerdo a las noticias v\u00eda\u00a0hokanko-alt<\/a>\u00a0y\u00a0Gematsu<\/a>, el ultimo Famitsu Semanal a confirmado que el lanzamiento del Hatsune Miku: Project Diva Future Tone para PlayStation 4, ser\u00e1\u00a0el d\u00eda 23 de Junio en Jap\u00f3n por\u00a0PlayStation network.<\/p>\n

La aplicaci\u00f3n principal necesaria para\u00a0lanzar el juego \u201cPrelude\u201d, estar\u00e1 disponible para descargar gratuitamente. Sin embargo, para a\u00f1adir canciones jugables y m\u00f3dulos, se requerir\u00e1 comprar uno (o ambos) de los siguientes paquetes de m\u00fasica a\u00a04212 Yenes: \u00abFuture Sound\u00bb con 120 canciones y enfoc\u00e1ndose en las selecciones musicales para las series Project Diva y Project Diva F, y \u00abColorful Tone\u00bb con 100 canciones enfocadas en las selecciones musicales Project Mirai y el Project Diva Arcade. Combinados, ambos paquetes ser\u00e1n de 340 modulos (trajes). Mas detalles en este articulo<\/a>.<\/p>\n

Esto implica que por 8424 Yenes (78$), se obtendra\u00a0pr\u00e1cticamente toda la selecci\u00f3n musical del Project Diva Future Tone con mas mejoras visuales que su contraparte \u00e1rcade. \u00a1No esta del todo mal! Los pedidos empezaran este mes, y aquellos que reservaron su compra, tendr\u00e1n derecho a dos temas para PS4 como bonificaci\u00f3n. \u00a1Tener en cuenta que el juego esta solo disponible desde\u00a0el PlayStation Network Japones!<\/p>\n
