Rangkuman Berita Otomachi Una selama Januari!

Kuroi Februari 2, 2018 Berita

Saatnya menuju ke bulan kedua 2018 dengan berita baik untuk Otomachi Una! Menjelang akhir Januari lalu, MTK mengungkapkan set kedua untuk LINE sticker Una bersama dengan konten baru di Niconico Commons page-nya!

LINE Sticker

Bagi mereka yang merasa belum cukup dengan set pertama LINE sticker yang menampilkan VOCALOID bertema belut ini, set keduanya, menampilkan Otomachi Una Talk Ex, kini tersedia di LINE Store seharga Rp 12,000!

Niconico Commons & Togetter Page

Sebelum merilis stikernya, sosial media Una menginformasikan fans mengenai konten baru Una di page mereka di Togetter, website content curation yang populer di Jepang. Di page ini, mereka mengumpulkan Tweet dan post dari Niconico Commons page untuk konten yang bisa didownload fans secara gratis dan digunakan di video Una mereka!

Ini adalah official page Otomachi Una untuk mendistribusikan official artwork, video material, wallpaper, dan data motion-capture! ♪є(｡´･◎･)｡´_ _)эﾍﾟｺﾘ
