Nanoblock, pencipta building block terkecil di dunia, mengumumkan perilisan dari nanoblock set spesial 10th anniversary, berdasarkan desain kostum Hatsune Miku V4X! Figure block yang manis ini cukup kecil untuk digenggam dengan telapak tangan saat selesai dirakit. Tambahan yang kecil namun kreatif bagi para fans Miku dan building block!<\/p>\n
