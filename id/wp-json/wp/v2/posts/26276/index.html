Sekali lagi, kami melakukan ekspansi lebih jauh dengan VNN. Namun kali ini kami menambahkan ProjectDIVA.wiki<\/a> dan ProjectDIVA.net<\/a> dibawah banner kami. Mereka tidak akan diimplementasikan langsung di website VNN, mereka tetap akan menjadi website yang berdiri sendiri. Mereka akan ditangani oleh system administrator kami Akurasix, dengan bantuan dari administrator lainnya dari website ini.<\/p>\n

Kami berharap penambahan ini akan memudahkan para user Project DIVA dari seluruh dunia untuk memiliki interface yang mudah digunakan dan sebuah tempat untuk seluruh informasi mengenai Project DIVA, baik itu lama atau baru. Kami akan berusaha yang terbaik di VNN untuk memastikan bahwa semua hal yang berhubungan dengan Project DIVA akan mudah diakses dan ditemukan oleh semua orang. Kami masih bekerja untuk meningkatkan beberapa aspek dari website ini, namun kami akan segera mengumumkan updatenya jika semuanya telah selesai dan siap.<\/p>\n
