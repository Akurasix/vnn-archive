\n

Hasil akhir dari kontes gambar untuk “Hatsune Miku: Project DIVA X End Credits” telah diumumkan. Kontesnya dibuat untuk dua Tema: “Kepada Hatsune Miku dan Kawan-Kawan, Terima Kasih \/ Dari Miku dan Kawan-Kawan, Terima Kasih” dan “(Konser) Live Idealmu”, dengan 100 ilustrasi untuk masing-masing tema, membuat kompetisi ini lebih intens dari kontes-kontes Piapro sebelumnya.<\/p>\n

The total submissions of 200 illustrations are now available on results announcement page<\/a> (kami tidak bisa menampung seluruh 200 ilustrasi tersebut disini). Para pemenang akan mendapatkan hadiah dari SEGA. Selamat kepada semua pemenang!<\/p>\n

Hatsune Miku Project DIVA X akan dirilis di Playstation 4 pada musim gugur 2016, dan PS VITA pada Bulan Maret 2016. Update saat ini mencakup setlist para producer dan ilustrator<\/a> dan update konten dari Tokyo Game Show 2015.<\/a>.<\/p>\n
