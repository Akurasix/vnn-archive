Kolaborasi antara IA<\/a> dan mobile game, Sapphire Sphere<\/a> ~ Blue Boundary ~ saat ini tengah berlangsung!<\/p>\n

Apa Itu Sapphire Sphere?<\/h2>\n

Sapphire Sphere adalah sebuah game mobile fantasy RPG dari X.D. Global<\/a> dan tersedia baik untuk device Android dan iOS.<\/p>\n

Apa Kolaborasinya?<\/h2>\n

Kampanye Sapphire Sphere x IA akan berlangsung hingga 29 Agustus.<\/p>\n

Selama kampanye ini, ‘Spirit Song IA’ akan tersedia sebagai karakter 5 star yang bisa diperoleh melalui in-game gacha!<\/p>\n

<\/a><\/p>\n

Karakter IA di game ini adalah Magician class dengan Wind attribute yang dapat menyerang musuh maupun memberikan buff kepada teman.<\/p>\n

Juga, selama periode kampanyenya, theme song pembuka gamenya akan dinyanyikan oleh IA!<\/p>\n

Untuk preview dari versi spesial theme song-nya, silakan cek video klip yang ada di Tweet berikut ini.<\/p>\n
