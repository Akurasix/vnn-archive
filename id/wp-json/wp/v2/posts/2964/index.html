Pengumuman terbaru hari ini untuk Hatsune Miku: Project Diva X telah mengungkapkan desain final untuk Box Art PS Vita version dari gamenya, juga sebuah Aime game card (ilustrasi oleh KEI) dimasukkan sebagai preorder bonus bersama dengan versi Jepang dari gamenya. Kartu ini dapat digunakan dengan Aime arcade game apapun, termasuk Project Diva Arcade!<\/p>\n
