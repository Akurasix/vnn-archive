Kalian harus teregistrasi dengan Vocalotracks untuk berpartisipasi (Kalian dapat melakukannya dengan mencari \u201cBox Kuning\u201d di bagian bawah dari website contest\u00a0page<\/a>). Registrasi harus memasukkan nama, alamat email dan nomor telepon.<\/p>\n

Peraturan Kontes:<\/strong><\/p>\n

Lagu:<\/strong><\/p>\n

    \n
  • Audio harus dalam format: .wav file\/16bit, 44.1kHz, STEREO.<\/li>\n
  • Ukuran file tidak boleh melebihi 65MB.<\/li>\n
  • Durasi lagu tidak boleh lebih dari 6 menit.<\/li>\n
  • Nama lagu harus dibuat dalam format: Katakana or huruf alfanumerik dan tidak boleh mengandung simbol khusus.<\/li>\n
  • Genre musik apapun diperbolehkan.<\/li>\n
  • Lirik harus dibuat dalam file .txt.<\/li>\n<\/ul>\n

    Gambar:<\/strong><\/p>\n

      \n
    • Gambar and artist image harus karya original, Gambar yang telah dilisensi atau kasar (violent) tidak akan diterima dan akan didiskualifikasi.<\/li>\n
    • Gambar harus disimpan sebagai .jpg file dan ukurannya: 2500 \u00d7 2500 pixels\/aspect ratio 1:1.<\/li>\n<\/ul>\n

      Hal Lainnya:<\/strong><\/p>\n

        \n
      • Informasi mengenai artist harus ditulis sebagai berikut: Pembuat Lirik dan Musik ditulis dalam huruf Katakana.<\/li>\n
      • Maksimal 3 lagu untuk setiap partisipan.<\/li>\n
      • Nama pemegang lisensi akan di-display di website Vocalotracks juga di website lainnya seperti\u00a0 amazon atau itunes.<\/li>\n
