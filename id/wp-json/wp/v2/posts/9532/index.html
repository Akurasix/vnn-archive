Sega telah mengumumkan<\/a> model kolaborasi Hatsune Miku dari Persona 4: Dancing All Night<\/a> untuk dirilis di Project DIVA X, Project DIVA Future Tone, dan Project DIVA Arcade Future Tone. Tiap gamenya akan memiliki tanggal perilisan yang berbeda-beda.<\/p>\n

Module kolaborasi ini akan dirilis sebagai DLC untuk Project DIVA X pada 23 Juni di PS Vita dan 25 Agustus di PS4 bersamaan dengan perilisan Project DIVA X HD<\/a> seharga 300 yen.\u00a0 Module ini akan termasuk dalam \u201cColorful Tone\u201d pack, satu dari dua pack<\/a> di Project DIVA Future Tone, yang akan dirilis bersamaan dengan perilisan gamenya di Japanese PlayStation Network tanggal 23 Juni<\/a>.<\/p>\n

Module ini juga akan tersedia di Project DIVA Arcade Future Tone untuk 300 VP (Vocaloid Points).<\/p>\n
