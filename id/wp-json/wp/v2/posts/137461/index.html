Sekarang sudah Musim Gugur, dan berarti kampanye musiman baru antara Hatsune Miku<\/a> dan Glico<\/a>! Berjudul “A Long Autumn Night’s Sweets Story” (\u79cb\u306e\u591c\u9577\u306eSWEETS STORY, Aki no Yonaga no SWEETS STORY<\/em>), kampanye baru ini merayakan datangnya Musim Gugur. Sama dengan kampanye sebelumnya<\/a>, setiap orang yang membeli sekurangnya dua snack Glico dari AEON<\/a> store yang berpartisipasi, mulai 26 September, akan menerima satu dari enam clear file folder yang berbeda, secara gratis! Ilustrasi untuk clear folder ini digambar oleh BUZZ<\/a>, \u3053\u3059\u305a\u3081 (Kosuzume)<\/a>, \u304b\u3074\u308a\u3053 (CATSCAT)<\/a>, \u30e8\u30b7\u30d5\u30df (Yoshifumi)<\/a>, \u677e\u3046\u306b (Matsuuni)<\/a>, dan U35 (Umiko)<\/a>!<\/p>\n
