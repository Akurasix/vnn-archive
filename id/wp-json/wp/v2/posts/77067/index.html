Sebuah bocoran dari penjual di Taobao<\/a> mengungkapkan SEGA akan merilis Prize Plushie Hatsune Miku baru pada 3 Februari 2017. Plushie ini berukuran jumbo, setinggi 30cm, dan dibuat berdasarkan modul kostum \u201cNatural\u201d, yang digambar oleh KEI untuk seri game Project Diva.<\/p>\n

Kalian dapat menemukan plushie ini buka preorder dari beberapa toko online, dengan jadwal perilisan sekitar bulan Februari 2017. Namun jika kalian kebetulan datang ke Jepang setelah 3 Februari (seperti untuk Snow Miku Festival), silakan coba memenangkannya di arcade center setempat!<\/p>\n
