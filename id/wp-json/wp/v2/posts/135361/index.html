Untuk para pengguna VOCALOID 4, tentunya menyenangkan untuk mencoba voicebank baru, walaupun hanya untuk waktu yang terbatas, terdapat banyak VOCALOID yang bisa digunakan untuk periode trial. Kini, Xin Hua V4 Japanese<\/a> juga tersedia free trial-nya di website Gynoid!<\/p>\n

Mengenai Versi Trial<\/h3>\n

Kami telah membicarakan V4 update-nya, dan kini kami berfokus secara khusus untuk Japanese voicebank baru Xin Hua yang akan dirilis tanggal 22 September 2017. Sementara menunggu perilisannya, Gynoid memutuskan untuk mengizinkan fans mendownload trialnya secara Gratis!<\/p>\n

Trial ini akan tersedia untuk 14 hari, dan diberikan bersama sebuah folder berisi ekstra file wav (breath file), bersama dua voicebank Xin Hua Japanese: Xin Hua Natural dan Xin Hua Power. Sama seperti VOCALOID4 lainnya, voicebank ini kompatibel dengan fitur Growl <\/em>dan dapat digunakan bersamaan berkat fungsi Cross-Synthesis <\/em>yang ada di V4 Editor.<\/p>\n

Untuk mendownload trialnya, klik disini<\/a> (tersedia versi Windows & Mac).<\/p>\n

Jika selama prosesnya, installer meminta product activation key, klik saja “Tidak”.<\/em><\/p>\n

Harap diingat bahwa kalian membutuhkan VOCALOID4 Editor untuk menggunakan trial ini. Jika kalian belum memilikinya, tidak perlu kuatir : Terdapat versi free trial software, yang dapat didownload di website YAMAHA setelah kalian registrasi akun disana. Kalian dapat menemukan trialnya disini<\/a>, setelah log in.<\/p>\n
