Puzzle game populer “Crash Fever” mengadakan event kolaborasi dengan Hatsune Miku dan VOCALOID Crypton lainnya. Kolaborasi Hatsune Miku x Crash Fever ini menampilkan Dua Unit Miku, Rin, Len, dan Luka (Original dan V4X) juga satu unit KAITO dan MEIKO. Bahkan juga terdapat unit Snow Miku dan Yukine!<\/p>\n

Event dimulai tanggal 3 Februari dan berakhir tanggal 17.<\/p>\n

Crash Fever tersedia untuk Android<\/a> dan iOS<\/a>.<\/p>\n

Event 3-10 Februari<\/h3>\n

Mulai 3~10 Februar di Global Ver. gamenya, fans bisa summon unit limited edition melalui “Hatcher”. Penarikan x10 yang pertama senilai 250 Polygon (mata uang gamenya yang bisa dibeli dengan uang sungguhan) sementara penarikan x10 berikutnya adalah 500 Polygon. Sekali penarikan adalah 50 Polygon. Namun, melakukan lima penarikan x10 menjamin kalian akan mendapatkan special unit. Selama periode waktu ini, user bisa mendapatkan 30 Polygon setiap harinya secara gratis.<\/p>\n
