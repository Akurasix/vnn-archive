Grup Tiongkok \u548f\u541f\u8f69 (Yong Yin Xuan)<\/a> adalah sebuah grup yang terdiri dari para artist Tiongkok (producer, pembuat lirik, illustrator, dll.). Anggotanya termasuk: JusF\u5468\u5b58<\/a>, \u8036Aya\u53c1<\/a>, Sya<\/a>, GhostFinal<\/a> dan banyak lagi!<\/p>\n

<\/a><\/p>\n

Mereka baru-baru ini merilis album berjudul “Play! Play!”, menampilkan 8 track menggunakan VOCALOID Tiongkok. Ketika saya (Monds) mendengarkan cross-fade albumnya, saya langsung tahu saya menginginkan album ini, saya terus mendengarkan cross-fade nya berulang-ulang sampai akhirnya saya menerima albumnya. Saya beritahu anda, saya tidak kecewa.<\/p>\n[bilibili src=4777133]\n

Package dan Bonus Merchandise<\/h2>\n

Saya membeli Deluxe package dari Taobao seharga ~$30 USD + biaya kirim. Harap diingat untuk mengirimkan barang dari Taobao dari Indonesia, anda akan membutuhkan jasa proxy \/ delivery service. Secara pribadi, saya (Kuroi) merekomendasikan anda membelinya lewat BuyChina, saya pernah beberapa kali membeli album dari sini, dan mereka menawarkan handling fee yang murah, dengan biaya kirim ke Indonesia\u00a0 yang juga sangat terjangkau, dengan beberapa alternatif paket pos. Namun tentunya kalian juga dapat menggunakan delivery service lainnya yang biasa kalian pakai.<\/p>\n
