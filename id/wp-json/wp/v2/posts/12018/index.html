Official English website <\/a>Hatsune Miku: Project DIVA X<\/a> telah dirilis hari ini. Websitenya kini menampilkan secara penuh berbagai detail dari yang versi Jepangnya ke bahasa Inggris, termasuk: introduction, Cloud Requests, music, modules, stages, concert editor, home, English trailers, dan berbagai game retailer yang terkait.<\/p>\n

Gamenya dijual seharga US$39.99 dan US$49.99 untuk versi PS Vita dan PlayStation 4, secara berurutan, dan saat ini telah menerima preorder di berbagai game retailer<\/a>.\u00a0Preorder akan mendapatkan sebuah bonus Hatsune Miku fabric pouch yang berhiaskan 5 aura symbol dari gamenya secara gratis.<\/p>\n

Hatsune Miku: Project DIVA X akan dirilis di North America tanggal 30 Agustus untuk PS Vita dan PS4<\/a>. Versi Jepang dari gamenya di PS4, Hatsune Miku: Project DIVA X HD, akan dirilis pada tanggal yang hampir bersamaan di 25 Agustus.<\/p>\n
