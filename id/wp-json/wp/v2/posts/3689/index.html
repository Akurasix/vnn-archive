Update berikutnya<\/a> untuk Hatsune Miku: Project DIVA X<\/a> hari ini telah diungkapkan! Update ini mencakup Free Play mode, Live Edit mode, dan lima lagu baru beserta modul kostumnya. Kita akan mulai dengan Free Play mode, juga dikenal sebagai judul baru untuk mode klasik Rhythm Game yang sudah sangat kita kenal.<\/p>\n

Free Play mode memiliki pola yang sama dengan titel-titel rhythm game yang sebelumnya, dimana kita dapat memainkan musik, stages, modul, dan aksesoris yang di-unlock di the Live Quest mode<\/a>. Ada beberapa detail yang penting yang perlu diperhatikan dalam mode ini: Module Drop<\/a> tidak akan muncul, efek skill module tidak berlaku, opsi transformasi Modul tersedia, dan Help &\u00a0Challenge item termasuk sebagai bagian dari fitur ini.<\/p>\n

Pada Live Edit mode, player diberi kebebasan untuk melakukan kustomisasi secara penuh dalam menciptakan rhythm game atau sebuah video, termasuk pilihan lagu, stage, modul kostum, mengatur berbagai sudut kamera, lighting, dll. Harap diperhatikan bahwa Live Edit mode akan tersedia sebagai free DLC di PlayStation store untuk PS Vita karena ukurannya yang sangat besar.Berikut adalah lima lagu baru dan modul kostumnya, termasuk opening theme song dari ryo(supercell)<\/a>.<\/p>\n
