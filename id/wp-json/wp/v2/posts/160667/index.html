Game ini sudah lama ditunggu-tunggu semenjak demonya diperlihatkan di Snow Miku 2017 yang lalu. Hatsune Miku VR akan dirilis untuk PC melalui platform Steam VR pada Musim Semi 2018, untuk Oculus Rift dan HTC Vive! Game ini dibuat oleh kolaborasi Crypton Future Media dan Degica Co.<\/p>\n

\u201cAction Rhythm Game\u201d ini menggunakan controller VR, dan memungkinkan kalian untuk menyaksikan Miku menari sementara kalian memainkan gamenya. Game ini akan dirilis dengan 8 lagu sebagai permulaan, dengan masing-masing 2 level kesulitan. Lagu tambahan telah direncanakan untuk kedepannya!<\/p>\n

Game ini tersedia dalam Bahasa Inggris, Jepang, Tiongkok, dan lain-lain. Informasi lebih detail akan diumumkan kemudian. Silakan cek official site Hatsune Miku VR disini! \u2192 http:\/\/promo.degicagames.com\/vr\/<\/strong><\/a><\/p>\n
