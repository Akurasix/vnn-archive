Miku Flick mobile app team dari SEGA mem-post berita di Twitter hari ini, mengumumkan pengakhiran layanan untuk Miku Flick\/02, salah satu dari app rhythm game mereka untuk iOS device. Tweet tersebut menyebutkan bahwa layanan gamenya akan berakhir di bulan ini tanggal 19 Juli. Alasan pengakhirannya mencakup tidak kompatibelnya app ini dengan device baru yang dirilis, perubahan pada billing API Apple, dan memfokuskan ulang kerja team SEGA.<\/p>\n
