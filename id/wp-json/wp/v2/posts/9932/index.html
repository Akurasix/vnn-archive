SEGA telah mengumumkan<\/a> sebuah game eksklusif untuk PlayStation VR berjudul \u201cHatsune Miku VR: Future Live\u201d, dan akan dirilis untuk PlayStation 4 di Jepang pada tanggal 13 Oktober 2016. Ini sepertinya dibuat berdasarkan SEGA 2015 E3 demo<\/a> yang sebelumnya diberi nama \u201cProject Morpheus\u201d VR set. Jangan salah mengartikan ini sebagai VR update untuk Hatsune Miku: Project Diva X, \u201cFuture Live\u201d adalah sebuah game independen untuk PlayStation VR!<\/p>\n

Game ini menampilkan mekanisme rhythm\/motion, yang diperlukan untuk mengisi sebuah voltage meter untuk membuat penonton bersemangat. Jika target jumlahnya tercapai, kalian akan bergabung dengan Miku di atas stage untuk sebuah penampilan spesial! Lebih banyak informasi mengenai game ini akan diumumkan kemudian.<\/p>\n
