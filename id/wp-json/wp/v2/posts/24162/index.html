Perubahan Harga<\/h3>\n

Fans Kagamine diluar Jepang harus membayar agak lebih mahal. Rin dijual seharga 70,000 yen (ditambah fee) sementara Len dijual seharga 75,000 yen (ditambah fee). Saat ini, belum ada informasi berapa jumlah fee tersebut.<\/p>\n

Cara Mendapatkannya<\/h3>\n

Saat ini, satu-satunya cara bagi para fans di luar Jepang untuk mendapatkannya adalah melalui web lottery.
