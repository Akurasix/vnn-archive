18 Desember adalah 3rd Anniversary VOCALOID MAIKA<\/a>! Untuk merayakan ulang tahunnya, Voctro Labs<\/a> mengadakan special anniversary sale hingga 22 Desember!<\/p>\n

<\/a><\/p>\n

Special Anniversary Sale<\/h3>\n

Hingga 22 Desember, fans dapat membeli software fisik (boxed version) MAIKA seharga 99\u20ac (Sekitar $104 USD) dengan mendapatkan boxed version Bruno<\/a> dan Clara<\/a> secara gratis! Boxed version Bruno dan Clara bernilai 109\u20ac (Sekitar $114 USD).<\/p>\n

Ini HANYA berlaku untuk pembelian software fisiknya. kalian tidak bisa mendapatkan secara gratis Bruno dan Clara dengan membeli versi download MAIKA.<\/p>\n

Sebagai tambahan, pembelian ini belum mencakup VOCALOID Editor. Untuk dapat menggunakan VOCALOID ini secara penuh, kalian harus membeli VOCALOID 3 atau VOCALOID 4 Editor secara terpisah. Namun mereka dilengkapi dengan Tiny VOCALOID Editor untuk kalian bisa langsung menggunakannya dengan fitur yang terbatas.<\/p>\n

Mengenai VOCALOID Tersebut<\/h3>\n

Bruno, Clara, dan MAIKA adalah VOCALOID Spanyol yang dibuat oleh Voctro Labs. Namun, MAIKA memiliki berbagai fonem yang memungkinkannya bernyanyi dalam beberapa bahasa. Demo resminya menampilkannya bernyanyi dalam bahasa Catalan, Italia, Inggris, dan Jepang. Ini membuat suaranya sangat fleksibel.<\/p>\n

Bruno dan Clara tidak dilengkapi dengan ekstra fonem tersebut, namun tetap menawarkan suara yang luar biasa. Dan dengan mendapatkan keduanya secara gratis sebagai bonus, kesempatan ini sangat sayang untuk dilewatkan! Ini adalah kesempatan bagi para pencinta \/ kolektor software fisik untuk mengoleksi Trio VOCALOID Spanyol ini dengan harga yang sangat murah!<\/p>\n

Cara Membeli<\/h3>\n

Kalian dapat membeli boxed & digital version MAIKA di website Voctro disini<\/a>. Kalian juga dapat mengecek Bruno dan Clara disini<\/a>. Halaman produknya juga menyediakan demo dari masing-masing\u00a0 VOCALOID tersebut, jadi pastikan kalian mengeceknya!<\/p>\n

Kalian juga dapat mengecek website VocaDB untuk lebih banyak lagu-lagu dari ketiganya.<\/p>\n
