LUMi Library Editions

There are three editions of LUMi currently available: Limited Edition, Regular Edition or a Download version.

Limited Edition
Price: 15,000 JPY

If you missed out on the chance to get the Limited Edition in person at AVA‘s office, or if you are simply an avid VOCALOID collector, then this edition might be the one for you.

The Limited Edition is presented in special packaging, featuring a white gift box made of washi and tied with a blue chord that is knotted into a pretty mizuhiki pattern.

If you are happy to spend some extra yen, it would be a lovely edition to any VOCALOID fan’s collection!

Regular Edition
Price: 12,000 JPY

The Regular Edition is the regular boxed version of the LUMi’s library and is priced in the mid-range.

Download version
Price: 10,000 JPY

As the name would suggest, this version is download only.

This is the cheapest option for VOCALOID users who are not set on getting the box.

System Requirements

For Windows
OS: Windows 10, Windows 8.1, Windows 8, Windows 7 SP1 (32/64bit)
CPU: Intel Dual Core CPU
RAM space: at least 2 GB (4 GB of free space is recommended)
Editor: VOCALOID4 Editor or VOCALOID4 Editor for Cubase

For Mac
OS: macOS Sierra, OS X El Capitan, OS X Yosemite, OS X Marvericks, OS X 10.8
CPU: Intel Dual Core CPU
RAM space: at least 2 GB (4 GB of free space is recommended)
Editor: VOCALOID4 Editor or VOCALOID4 Editor for Cubase
