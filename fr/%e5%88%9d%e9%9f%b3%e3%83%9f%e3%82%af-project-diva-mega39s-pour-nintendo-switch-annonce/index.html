初音ミク Project DIVA MEGA39’s pour Nintendo Switch Annoncé!

marklinmax juillet 1, 2019 Actualités

Pour célébrer le 10ème anniversaire du SEGA feat. HATSUNE MIKU Project, SEGA a annoncé le nouveau jeu 初音ミク Project DIVA MEGA39’s (prononcé Mega Mix) qui sortira sur Nintendo Switch début 2020!

Les chansons

Le nouveau jeu inclura 101 chansons. L’une d’entre elles sera composée juste pour le jeu, 10 seront nouvelles, et 90 viendront des jeux précédents! Jusque-là SEGA a révélé deux des nouvelles chansons : Alien Alien de Nayutalien 39 Music! de MikitoP

Nouveau mode de jeux

En plus du mode rythme standard des jeux Project DIVA, Project DIVA MEGA39’s aura un tout nouveau mode rythme! Les joueurs seront libres de changer de mode, comme le bouton Project mirai et le mode tactile. Les détails de ce nouveau mode n’ont pas encore était révélés, mais il sera jouable au Magical Mirai 2019.

Changements de style visuel

Project DIVA MEGA39’s offrira aussi un look complètement refait. Les clips vidéos des musiques ont maintenant un style « animé ». C’est nouveau et frais en même temps!

 

Concours de design Piapro

En plus du reste, SEGA et Crypton organiseront un concours de design sur Piapro! Le thème de ce concours est T-shirt costume design, et les gagnants verront leurs costumes inclut dans le jeu. Le concours commencera le 7 juillet et durera jusqu’au 15 aout!

DLC Chansons pour PS4

【お知らせ】PS4『初音ミク Project DIVA Future Tone/DX』をお持ちの方に向け『初音ミク Project DIVA MEGA39's』からの新規収録楽曲等をセットにしたDLCを、2020年に有償配信予定です。（アニメ調PV表現へのリニューアルや新しいゲームモードは「MEGA39's」ならではの要素です。） #pjd_sega

— SEGA feat. 初音ミク 公式 (@pjd_sega) July 1, 2019

Finalement, les nouvelles chansons pour Project DIVA MEGA39’s sortiront en DLC avec Project DIVA Future Tone et Project DIVA Future Tone DX en 2020. Cependant, le nouveau style visuel et le nouveau mode de jeu seront exclusifs a Project DIVA MEGA39’s, donc si vous voulez profiter de ceux-ci, assurez-vous de vous le procurer! Par-dessus tout cela, SEGA a mentionné qu’ils ont encore plus de choses à nous faire découvrir pour le 10ème anniversaire de la série Project DIVA! Restez attentifs!

Vous pouvez trouver le SEGA feat. HATSUNE MIKU project sur leur site officiel, Twitter et Youtube. Hatsune Miku est sur Facebook (Anglais et Japonnais), Twitter (Anglais et Japonnais), Youtube et Weibo. Allez aussi voir le site officiel, Twitter du Magical Mirar 2019!
