ProjectDIVA.net est un forum qui \u00e9tait pr\u00e9vu \u00e0 la base seulement pour les jeux Project Diva<\/em>.<\/p>\n

R\u00e9cemment, nous avons fait plusieurs changements sur le site.<\/p>\n

<\/a><\/p>\n

<\/a><\/p>\n

Vous pouvez aller sur le site \u00e0 ProjectDIVA.net<\/a> et devenir membre !<\/p>\n

Note : Nous devons valider tous les membres \u00e0 joinder le site pour \u00e9liminer les spams. Donc si vous ne pouvez pas commenter tout de suite, donnez-nous 24 heures pour tout v\u00e9rifier.<\/em><\/p>\n

Nous avons aussi une section sous \u00ab\u00a0Forum international<\/a>\u00a0\u00bb pr\u00e9vue pour les autres langues. Pour l\u2019instant, nous avons toutes les langues disponibles sur VNN. Beaucoup d\u2019autres mises \u00e0 jour arriveront plus tard, donc restez connect\u00e9s !<\/p>\n

ProjectDIVA.wiki<\/strong><\/a><\/p>\n

Le ProjectDIVA.wiki est juste ce qu\u2019il a l\u2019air d\u2019\u00eatre, un wiki pour les jeux o\u00f9 vous pouvez trouver la liste des chansons, les succ\u00e8s, les modules, et plus.<\/p>\n

Quoi de Neuf ?<\/strong><\/p>\n

Tout d\u2019abord, nous avons fait quelques changements dans le syst\u00e8me de navigation pour le rendre moins d\u00e9sordonn\u00e9 et plus facile d\u2019utilisation. Nous travaillons aussi \u00e0 l\u2019ajout de tous les jeux mentionn\u00e9s plus haut. Cependant, pendant ce temps, vous pouvez toujours trouver plein d\u2019informations sur ceux d\u00e9j\u00e0 pr\u00e9sents.<\/p>\n
