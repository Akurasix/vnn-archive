Une VOCALOID f\u00e9minin cr\u00e9\u00e9e par Akatsuki Virtual Artist (AVA) nomm\u00e9e LUMi a \u00e9t\u00e9 annonc\u00e9e! Elle est doubl\u00e9e par la c\u00e9l\u00e8bre doubleuse\u00a0Ohara Sayaka<\/a>, connue pour ses r\u00f4les d’Erza Scarlet (Fairy Tail<\/em>) et\u00a0Sailor Neptune (Sailormoon Crystal<\/em>). Le d\u00e9sign du personnage est illustr\u00e9 par Fugin Miyama, un artiste bien connu pour les s\u00e9ries Overlord<\/em>\u00a0et\u00a0Eureka Seven<\/em>. LUMi a \u00e9t\u00e9 confirm\u00e9 comme VOCALOID 4.<\/p>\n

Pour le moment, nous n’avons pas beaucoup d’informations sur elle, autres qu’il y aura une version d’essai qui sortira le 3 juin 2017. Aucune d\u00e9mo de chanson ou d’\u00e9chantillon vocal n’est sortie. L’annonce est disponible sur le site web officiel d’AVA<\/a>, en plus d’un tweet du Twitter d’Orihara Sayaka.<\/p>\n
