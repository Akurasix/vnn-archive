Pr\u00e9parez-vous, fans d\u2019outremer\u00a0! Les inscriptions sont maintenant ouvertes (22 avril) pour obtenir des tickets pour l\u2019\u00e9v\u00e9nement live Hatsune Miku: Magical Mirai 2017!<\/p>\n

Chaque personne peut s\u2019inscrire pour obtenir jusqu\u2019\u00e0 2 tickets par spectacle, avec 5 spectacles au total. Les options de places incluent aussi la possibilit\u00e9 d\u2019avoir une place en section SS ou S. Les inscriptions pour les tickets resteront ouvertes jusqu\u2019au 22 mai \u00e0 23h59 heure japonaise. Apr\u00e8s cela, les gagnants des inscriptions seront contact\u00e9s concernant les places qu\u2019ils ont ou n\u2019ont pas gagn\u00e9 selon l\u2019attribution al\u00e9atoire.<\/p>\n

Page d\u2019inscription pour les tickets: https:\/\/tixeebox.tv\/lp\/magicalmirai2017-entry-en\/<\/a><\/p>\n

Pour ceux qui attendent d\u2019avoir un logement sur place avec un ticket de classe SS, les inscriptions commenceront le 24 avril \u00e0 12h heure japonaise sur cette page: http:\/\/entame.knt.co.jp\/2017\/jimagicalmirai\/en\/<\/a><\/p>\n
