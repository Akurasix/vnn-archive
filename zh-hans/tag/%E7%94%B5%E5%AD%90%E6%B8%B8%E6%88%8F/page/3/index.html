    PS4版《初音未来-Project DIVA-Future Tone》6/23日本首发

    根据 hokanko-alt 和 Gematsu 的报导，最新一期的Fami通周刊已证实PS4版《初音未来-Project DIVA-Future Tone》将在6月23日上架日本地区的PlayStation平台。 启动游戏的核心APP “Prelude”可免费下载。但在游戏内添加曲目或模型则需要额外购买曲包，曲包选项有： “Future Song”曲包：内含120首Project Diva和Project Diva F系列乐曲。 “Colorful Tone”曲包：内含100首Project Mirai和Project Diva Arcade系列乐曲。 以上两个曲包总共有340种人物模型（服装），各个曲包售价为4,212日元。更多游戏详情可见上一篇报导。（英文注意） 这也表示只需8424日元（$78美元），你就可以在大幅度提升视觉效果的条件下畅玩Project Diva Arcade Future Tone 的所有曲目了，想想就觉得令人心动啊！预售将在这个月开启，提前购买的玩家们将可获得额外两个PS4主题。注意：游戏只在日本地区的PlayStation抢先上架！ 原文: Mikufan

    初音未来与手游《魔法使いと黒猫のウィズ》联动！

    初音未来与他的小伙伴们将确定联动手游《魔法使いと黒猫のウィズ》(Android/iOS)。此次联动将邀请著名P主DECO＊27 和 MitchieM为游戏作曲，还有由画师 KEI， iXima， たま (Tama)以及 BUZZ特别绘制的卡牌哦！ 从名字看来，游戏玩法是由回答一系列小测试来发出攻击。玩家将扮演魔法师的角色，以正确回答问题来发动攻击技能。卡牌角色属性将决定给敌人造成的伤害值。 DECO＊27本人也会作为特别嘉宾出现在10月26日21:00(日本时区)的Nico生放送里。观众将会在生放送里一睹为这次联动特别制作的PV，PV将随后在官方网站放出。 联动将由10月30日 16:00 （日本时区）开始至11月30日 15:59 （日本时区）, 只限于日服。                                                         - 原文来自:Mikufan.com

    初音未来 : Project DIVA X 片尾画作募集获奖名单发表！

    初音未来 : Project DIVA X的片尾鸣谢画作募集的最终结果发表啦！画作征集分为两个主题： “致Miku与她的小伙伴们的感谢”/“来自Miku与她的小伙伴们的感谢” 和 “理想中的live（演唱会）”，每个主题最终将各采用100个作品，令这次的piapro赛事空前竞争激烈。 共200副最终入选画作皆被收录在结果发表页面（200张图本站的确hold不住抱歉了）。获胜者将会获得由SEGA提供的奖品。恭喜各位入选的画师！ 初音未来 : Project DIVA X 将会在2016年的秋天上架PlayStation 4 以及 2016年3月上架PS Vita。 目前已知游戏更新包括 已收录producers & 画师名单 和 在Tokyo Game Show 2015上发表的新增游戏内容. - 原文来自：Mikufan.com
