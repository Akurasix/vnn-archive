    让Unity-chan跟着你说话：Voice Changer

    你可否想过让Unity-chan说些什么话，却苦于不会操作 Unity引擎？该怎么让她用高音说话或是让她听起来男性化？ @kaigyoPG做到了这一点。由于这是一套独立的程序，操作时并不需要使用Unity引擎。 https://twitter.com/kaigyoPG/status/694584021016190976 推特上的下载地址 「做了个Unity的Voice Changer Filter的演示版应用。你们可以从连结下载。Unity-chan还会跟着她说的话变化口型。」 只要透过麦克风，Unity-chan就会复诵你所说的每一句话、笑声、甚至是键盘的敲击声。程序介面里也有一个Unity-chan朗诵「welcome to the aperture science computer aided enrichment center」的音声演示，若你不满足于Unity-chan的默认声线，那么你可以变更音高及音频任一项参数来改变她的声音。如果这都不够你玩，还有两项预设设置可变更。 透过左栏「Face Update」的选项来变更Unity-chan的脸部表情。也可以透过右上角的「Change Motion」或空白键来变换Unity-chan的动作。目前这套程序仍为演示版本，希望未来将会有完整版本释出。

    Unity-chan与AKAZA的最新消息释出!

    许多关于Unity-chan与PROJECT AKAZA的消息已于Comiket宣布。AKAZA是否就是Unity-chan的VOCALOID角色名这件事仍有待商榷。目前也公开了几个Unity-chan还有VOCALOID SDK for Unity。 VOCALOID SDK for Unity演示版 官方的VOCALOID网站提供三个不同的项目给人下载，但我们目前不大确定第三个项目有何作用。 VOCALOID SDK for Unity 第一个项目是在Unity软件上操作Unity-chan声库的VOCALOID SDK for Unity。使用之前必须先安装5.2.2版本以上的Unity软件才能使用Unity编辑器和声库来制作歌曲。 Unity软件与Unity编辑器可 于此下载。 数据亦可导入VOCALOID编辑器使用，然而该声库却不支持V4的Growl功能。根据几名P主所言，这是因为声库是基于V2引擎制作的。这意味着她亦可在V2编辑器中被使用。 Unity Sample Project同捆版 VOCALOID SDK for Unity 「Unity-chan Live舞台！」Candy Rock Star——VOCALOID版本 第二个项目看来是可以利用Candy Rock Star的MV并且调教这首DEMO曲的歌声。 https://youtu.be/V7Ku0O7h9S4   UNITY WITH VOCALOID实时演唱 该教程视频提供如何使用K-shi工作室释出的演示程序「UNITY WITH VOCALOID」。 这项演示旨在利用几项简单的参数测试Unity-chan的声库。只需在文字框内输入文本，接着使用虚拟键盘播放。这个程序主要提供给那些计算机不足以运行Unity软件以及Unity编辑器的使用者们，使他们可以不必为了使用Unity-chan的声库而对计算机造成太大的负担。 Project AKAZA 有趣的是，最新消息似乎表示Unity-chan与神秘少女AKAZA有某种关联。我们不敢一口咬定，但根据上传到YouTube的Comiket DEMO曲，AKAZA的声音提供者是Unity-chan的声音演员，角元明日香小姐。 AKAZA在Comiket DEMO曲中作为VOCALOID登场 在这首DEMO曲中，Unity-chan的声音仍由角元明日香小姐提供，取而代之作为VOCALOID登场的却是AKAZA。 尽管如此，官方网站的下载页面仍显示声库为Unity-chan。 虽然现在我们还掌握不到太多消息，但请随时关注我们以取得最新信息！
