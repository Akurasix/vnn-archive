    PS4版《初音未来-Project DIVA-Future Tone》6/23日本首发

    洛天依将出席Bilibili Macro Link 2016！

    Bilibili弹幕网站的年度线下Live活动——Bilibili Macro Link（简称BML）即将举办第四届活动。 往年的BML都会邀请到B站内的著名唱见、舞见、音乐人，甚至是三次元艺人等作为出场嘉宾，而这届的BML也在众人的期盼下公布了2016年的出场嘉宾名单。而今年这位出道四年、已俘获无数B站观众的二次元歌姬——洛天依也不负粉丝所望地被列在嘉宾名单里，这也是BML开办至今第一次有Vocaloid全息投影演出。   2016 Bilibili Macro Link宣传海报 出场嘉宾介绍 今年的BML举办日期为7月23日，地点是上梅赛德斯奔驰文化中心（上海）。目前得知洛天依将会和日本虚拟偶像ミス・モノクローム（黑白小姐）以全息投影的方式同台表演，演唱曲目仍未公布。 更多BML 2016出场嘉宾名单请见这里。 票务相关 Bilibili Macro Link 2016共分6档票种，分别为： S级：1280元（人民币） A级：980元（人民币） B级：780元（人民币） C级：580元（人民币） D级：380元（人民币） E级：180元（人民币） Bilibili和大麦网将在5月20日同时开放购票通道。 通过Bilibili购票通道可购买S、A席门票（需为注册会员/正式会员）。 通过大麦网购票通道可购买B、C、D、E席门票（需为大麦网注册用户）。 每位用户最多可购买所有级别的门票共4张。   感兴趣的各位可以进入2016 Bilibili Macro Link主页以获得最新BML信息以及往届BML回播视频。
