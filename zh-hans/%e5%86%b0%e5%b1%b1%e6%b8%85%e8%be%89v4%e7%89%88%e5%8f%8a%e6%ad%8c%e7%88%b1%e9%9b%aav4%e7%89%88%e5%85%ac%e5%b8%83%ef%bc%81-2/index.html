冰山清辉V4版及歌爱雪V4版公布！

himmeltengoku 9月 30, 2015 新闻

 

 

AHS（A社）刚刚发布了Vocaloid声库“冰山清辉”及“歌爱雪”V4版本的发售日，官方试听曲以及最终人设。“冰山清辉”的V4版将以平常老师版（Natural）及摇滚版（ROCK)两种不同声库来发卖，全套将可以单独购买，也将有与VOCALOID4编辑器打包的套装形式可供选择。歌爱雪的V4版将会很残念的只有Natural版一种，但同样也有与V4编辑器捆绑的套装版发售。

“冰山清辉”及“歌爱雪”两款声库的实体版与电子版的预定发售日为2015年10月29日，但与V4编辑器捆绑的套装版本将只会有实体发行。

冰山清辉 V4 Natural

冰山老师的V4版设计被流出已经有一段时间了，老师的穿着很有一股“自然”“休闲”的感觉呢。

之前有购买过冰山老师V2版的小朋友们将会可以以折扣价进行V4版升级。另此之外，还会有一个“A社用户特别版“，我们现在还搞不太清楚这个”特别版“里面会有什么，但是据官方宣称该“特别版”将会仅提供给拥有A社产品并且可以提供验证码的用户。

与V4编辑器打包版: 10,800 日元 + 消费税

下载版: 8,000 日元 + 消费税

V2 升级版: 6,000 日元 + 消费税

AHS 用户特别版: 8,500 日元 + 消费税

冰山老师的Natural版的音域将在A1到C4之间，推荐BPM将在60到160之间.

到目前为止冰山老师Natural版的官方MMD模型还未在计划当中。尽管V2版是有模型的，如果未来V4版也会公布MMD的话，也许可以在这里下载的到哦。

 

 

 

 

 

 

 

冰山清辉 V4 ROCK

冰山老师的V4 Rock版单独售价将会和Natural版一样，但其他打包版的价格将会略贵一些。

双声库打包版

与V4编辑器打包版: 17,800 日元 + 消费税

V2 升级版: 10,800 日元 + 消费税

AHS 用户特别版: 14,800 日元 + 消费税

冰山老师V家新手包

与V4编辑器打包版: 23,800 日元 + 消费税

V2 升级版: 16,800 日元 + 消费税

AHS 用户特别版: 20,800 日元 + 消费税

冰山老师的Rock版的音域将在C2到G4之间，推荐BPM将在70到190之间.

另外，A社还公布了一段由OzaShin制作的同时使用冰山老师双版本声库的示范曲。

 

 

 

 

 

 

 

歌爱雪 V4
如我们刚刚所提到的，歌爱雪的V4版将只有Natural一种版本。她的最佳音域将在F2到C4之间，建议BPM在50到150之间

她的新造型将会在发布时有一个MMD模型随之配布。

在此之外，她将也会有V家新手包版本捆绑发售。

歌爱雪 Natural

与V4编辑器打包版: 9,800 日元 + 消费税

下载版: 7,000 日元 + 消费税

V2 升级版: 5,000 日元 + 消费税

AHS 用户特别版: 8,000 日元 + 消费税

歌爱雪 V家新手包

与V4编辑器打包版: 16,000 日元 + 消费税

V2 升级版: 11,000 日元 + 消费税

AHS 用户特别版: 14,000 日元 + 消费税

 

 

 

 

 

歌爱雪V4的第一支官方示范曲的作者是DAISUKE-P
