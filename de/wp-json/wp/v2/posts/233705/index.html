Das Project Diva-Team von SEGA hat eine 7-t\u00e4gige Umfrage auf Twitter gestartet, um Hatsune Miku-Fans nach der Ver\u00f6ffentlichung eines Hatsune Miku: Project Diva-Controllers im Arcade-Stil f\u00fcr PlayStation 4, Nintendo Switch oder beides, zu befragen!<\/span><\/span> Die Umfrage wurde wahrscheinlich durch den bevorstehenden Launch von Project Diva: MEGA 39’s f\u00fcr die Nintendo Switch<\/a> ausgel\u00f6st.<\/span><\/span><\/p>\n

Fans k\u00f6nnen bei der Umfrage abstimmen, f\u00fcr welche Konsole sie einen Controller kaufen m\u00f6chten.<\/span> Bitte stimme nur ab, wenn du dir den Controller tats\u00e4chlich kaufen w\u00fcrdest!<\/span> Sie wollen nicht, dass die Ergebnisse von begeisterten Fans verzerrt werden.<\/span> Die folgenden Optionen sind aufgef\u00fchrt:<\/span><\/span><\/p>\n

1: Ich m\u00f6chte eine PS4-Version kaufen.<\/span>
\n2: Ich m\u00f6chte eine Switch-Version kaufen.<\/span>
\n3: Ich m\u00f6chte beide Versionen kaufen.<\/span>
\n4: Ich werde keine der Versionen kaufen.<\/span><\/span><\/p>\n

Das Project Diva-Team gibt an, dass die aufgelisteten Produkte in Betracht gezogen werden und die Gewinnerauswahl f\u00fcr die Ver\u00f6ffentlichung im Jahr 2020 erwartet wird. Gib deine Stimme f\u00fcr den folgenden Tweet ab!<\/span> Die Umfrage endet am 19. Juli um 19:00 Uhr JST.<\/span><\/span><\/p>\n
