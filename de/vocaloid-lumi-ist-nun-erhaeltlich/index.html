LUMi Library Editionen

Momentan sind drei verschiedene Versionen von LUMi erhältlich: Limited Edition, eine normale Edition oder ein Download.

Limited Edition
Preis: 15.000 JPY

Falls du deine Chance verpasst hast an die limitierte Edition im Büro von AVA zu gelangen oder einfach nur ein gnadenloser VOCALOID-Sammler bist, dann ist diese Edition das Richtige für dich!

Die Limited Edition wird in einer speziellen Verpackung vorgelegt mit einer weißen Geschenkverpackung aus Washi (Japanpapier) und einem blauen Bändchen in einem Mizuhiki-Muster zusammengeknottet.

Falls du gern etwas mehr Yen ausgeben willst, wäre es eine entzückende Edition für jede VOCALOID-Sammlung!

Normale Edition
Preis: 12.000 JPY

Die normale Edition ist die reguläre Box-Version von LUMis Library und befindet sich im mittleren Preisbereich.

Download Version
Preis: 10.000 JPY

Wie der Name schon vermuten lässt, handelt es sich hierbei um einen reinen Download.

Das ist die günstigste Variante für VOCALOID-Nutzer, die nicht an einer Box interessiert sind.

Systemanforderungen

Für Windows
Betriebssystem: Windows 10, Windows 8.1, Windows 8, Windows 7 SP1 (32/64bit)
Prozessor: Intel Dual Core CPU
Arbeitsspeicher: mind. 2 GB (4 GB empfohlen)
Editor: VOCALOID4 Editor oder VOCALOID4 Editor für Cubase

Für Mac
Betriebssystem: macOS Sierra, OS X El Capitan, OS X Yosemite, OS X Marvericks, OS X 10.8
Prozessor: Intel Dual Core CPU
Arbeitsspeicher: mind. 2 GB (4 GB empfohlen)
Editor: VOCALOID4 Editor oder VOCALOID4 Editor für Cubase

Kostenlose Testversion

Für diejenigen, die sich noch nicht sicher sind, ob sie sich LUMi kaufen sollen, gibt es die Möglichkeit sich hier als Mitglied zu registrieren und die kostenlose Testversion herunterzuladen.
