みんなはもう食べた？初音ミクの限定コラボロールケーキが秋葉原と新宿で発売中！（※追記あり）

kowhey 8月 25, 2016 ニュース, 感想

PS4向けにゲームソフト「初音ミク Project DIVA X HD」が発売されましたが、それに関連して、KEIさんの描いたメインビジュアルをもとにした初音ミクの限定ロールコラボケーキが東京の秋葉原と新宿で販売が行われています。

今回は「初音ミク Project DIVA X HD」と東京で2店舗のスイーツショップを展開する「スウィートパラダイスケーキショップ」とのコラボレーション企画で、DIVAXにちなんだコラボスイーツが期間限定で3種類登場しています。メインビジュアルの初音ミクさんをそのままケーキにプリントした「限定ロールケーキ / 1800円」、様々な味がミックスされをいろんな味を楽しむことができる「ポップコーン / 800円」、各キャラクターをイメージした「コラボドリンク / 500円」。

なお、「ロールケーキ」を購入した方には特大缶バッチ、「コラボドリンク」を購入した方には普通サイズの缶バッチが特典としてついてきます。

ちなみに9月30日まで販売を実施しているということなので、「初音ミクシンフォニー」の日はもちろん「マジカルミライ」の帰り際に秋葉原に寄って食べることもできます。ただし、8月25日は約1時間で1日分が完売してしまうほどの人気だったというツイートも見られたので、なるべく早い時間に行った方がコラボケーキを購入しやすいかもしれません。

キャラクタークレープも登場！（追記）

さらに、初音ミク公式ブログによれば、東京・渋谷にあるキャラクタークレープ「渋谷マルイ」（スイーツパラダイスとは違う店舗です！）で、それぞれのキャラクターをイメージした「ミククレープ」というコラボ商品も登場！こちらはコラボロールケーキとは別のコラボ企画で、販売日は8月26日～9月8日まで！詳細は初音ミク公式ブログの記事をご確認ください。

販売場所

「初音ミク Project DIVA X HD コラボレーションロールケーキ」は以下の2店舗で販売中！実際に食べてみたいと思った方はこちらの地図を確認しておきましょう！海外から秋葉原や新宿へ寄っていく予定のある方は是非どうぞ！

    2. スイーツパラダイスケーキショップ・新宿メトロ店
      東京都新宿区西新宿1-1-2　新宿地下鉄ビルB2Ｆ　メトロ食堂街内
